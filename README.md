# button-reader
Arduino Project: Learning pushbuttons and serial monitors

This was much harder than initially anticipated. I learned the very basics of Arduino wiring here, that electricity only flows from 5V to GND. Pins have an input mode where they sense electricity and an output mode where they send it out. On a button, the legs that face towards each other are a pair, and when it is pressed, the two pairs are connected. A resistor is necessary because sending 5V directly to GND is harmful to the Arduino. Learning these little things and the code that came with them was a bit painful even with a tutorial.
